Geração de Computadores

(Voltar)

1° Geração (1945/1955)

  • Computadores a válvula
  • Ausência de um Sitema Operacional
  • Programação em linguagem de máquina (bits 0 e 1)
    Ex: Colossus Mark I, ELetronic Numerical Integrator and Computer (ENIAC)

2° Geração (1955/1965)

  • Invenção e uso dos Transistores
  • Uso da linguagem Assembly e FORTRAN
  • Sistemas Operacionais do tipo lote (batch)Executa vários programas um atrás do outro

3° Geração (1965/1980)

  • Circuitos integrados
  • Multiprogramação
  • Time-Sharing
  • Sistema OS/360(IBM)1° a usar circutio SSI

4° Geração (1980/1990)

  • Circuitos integrados com alta escala de integração (LSI - Largel Scale Integration)
  • Sistema Operacional para micros (MS- DOS e Windows)
  • Difusão da Internet

5° Geração (1990/Hoje)

  • Difusão da Internet
  • Sistema Operacional com suporte para TCP/IP
  • Client/Server
  • Sistemas de tempo real
  • Computação Ubíqua
  • Internet das Coisas
  • CPs Cyber Physical System

Arquitetura de um Computador

(Voltar)

  • Arquitetura são atributos visíveis ao programador:
    • Impacto direto sobre a execução lógica de um programa;
    • Conjunto de instruções, números de bits usados para representação de dados, mecanismos de E/S, técnicas de endereçamento.
  • Organização é como os recursos são implementados:
    • Unidades Operacionais e suas interconexões que realizam suas especificações da arquitetura
    • Detalhes do Hardware são transparentes ao programador
    • Sinais de controle, interfaces e tecnologias de memória

Tipos de Redes

(Voltar)

  • Mainframe: É um computador de grande porte dedicado ao processamento de grandes volumes de dados, com alto desempenho, performance, escalabiliade e segurança.
  • MAN = Metropolitan Area Network: Redes de áreas metropolitanas
  • WAN = Wide Area Network: Redes de longa distância

Endereços IP Classes A, B, C, D e E

(Voltar)

Para acomodar redes de diferentes tamanhos e ajudar na classificação dessas redes, considerando também a quantidade de octetos associados a quantidade de redes e hosts, criou-se as classes de IPs:

  • Classe A: São atribuidos a redes maiores
  • Classe B: São atribuidos a redes de porte médio
  • Classe C: São atribuidos a redes pequenas
  • Classe D: São atribuidos a grupos de Multicast:
    • É um metódo ou técnica de transmissão de um pacote de dados para múltiplos destinos ao mesmo tempo
  • Classe E: São reservados apenas para pesquisas
